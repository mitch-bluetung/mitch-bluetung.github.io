Microprocessor Applications project board.
An ATMEGA32L microcontroller and EPM7032AE programmable logic device were used to take input from a 4x4 keypad, decode the signal and output to a 7-segment display.
Quartus II (Verilog HDL) was used to program the logic blocks for the PLD, and ATMEL Studio (C) was used for the microcontroller.
Full build documentation and report.

Digital Signal Processing project board.
A project board designed to take an analogue input, interface with an external DSP controller that filters the signal to programmed specifications, then interface back to the board for analogue output.
Coded using Assembly and C. MATLAB was also used to design the filter whose coefficients were then implemented in the C program.
Full build documentation and report.
